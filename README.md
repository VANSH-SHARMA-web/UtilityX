# UtilityX — Smart Tools. Essential Services.

A fast, minimal, all-in-one utility website built with pure HTML, CSS and JavaScript.
No frameworks. No dependencies. Just tools that work.

## 🛠 Tools Included
- 🔢 Mini Calculator
- 🔬 Scientific Calculator
- 📏 Unit Converter (Length, Weight, Temp, Volume, Speed, Area, Data)
- 🔐 Password Generator with strength meter
- 💻 Base Converter (Binary, Octal, Decimal, Hex)
- 🎨 Color Converter (HEX, RGB, HSL)
- ⚖️ BMI Calculator
- ⏱ Stopwatch with lap tracking
- 📝 Text Tools (9 transforms + stats)
- 🎂 Age Calculator

## ✨ Features
- Light / Dark mode
- Last 3 results history panel
- Backend code examples in C++ and Python
- Fully responsive
